# Changelog

本项目使用语义化版本管理：`v主版本.次版本.修订版本`。

- 修订版本：修复 bug、小 UI 调整、不改变核心功能，例如 `v1.1.2 -> v1.1.3`。
- 次版本：新增一组完整功能，例如 `v1.1.2 -> v1.2.0`。
- 主版本：大规模架构调整或接口不兼容，例如 `v1.x.x -> v2.0.0`。

## v1.1.5

- 完成 390px、430px、768px 移动端最终巡检。
- 个人资料页移动端背景改为轻量渐变，减少 emoji 背景渲染开销。
- 编辑个人资料页补齐移动端适配，头像上传、密码修改和 OJ 绑定在窄屏下改为纵向布局。
- 平台统计明细弹窗打开时锁定背景滚动，弹窗内部独立滚动。
- 优化成就抽屉、团队编辑、消息页和统计排行榜在小屏下的间距、按钮尺寸和滚动区域。
- 成就详情长列表启用延迟渲染，降低抽屉打开后的渲染压力。
- 左下角状态栏版本号更新为 `v1.1.5`。

## v1.1.3

- 确认弹窗迁移到统一按钮组件，减少页面内重复按钮样式。
- 新增 `design-system.css` 全站兼容层，统一历史按钮、分页、消息和团队操作按钮的视觉标准。

- 新增 API 错误解析工具，统一读取后端 `message/error/reason/code`。
- Toast 增加空消息兜底，避免成功或失败提示为空。
- 左下角状态栏版本号更新为 `v1.1.3`。
